Java MP3 Plugin

Softsqueeze use a Java Sound plugin to play MP3 audio files. The jlayer MP3 plugin is included with Softsqueeze and can be used 'out of the box'.

An alternative plugin, the Java MP3 Plugin, can also be used. This plugin is recommended because it uses less cpu to decode the MP3 audio files. Due to licensing restrictions it cannot be included with Softsqueeze and you will need to download and install it. Once installed you can choose which MP3 Plugin to use in the Softsqueeze Audio Properties.

Download the Java MP3 Plugin, then install it using the appropriate instructions below.

Windows

Download and run the Windows Platform .exe file. This installer should automatically locate your Java program files, and install the plugin.

Some people have had trouble with the installer not working. In this case download the Other Platform .zip file. Unzip the file and copy the mp3plugin.jar file into your Java lib/ext folder, for example C:\Program Files\Java\jdk1.5\jre\lib\ext.

OS X

Download the Other Platform .zip file. Unzip the file and copy the mp3plugin.jar file to /Library/Java/Extensions/.

Linux

Download the Other Platform .zip file. Unzip the file and copy the mp3plugin.jar file into your Java lib/ext folder, for example /usr/java/jdk1.5.0/jre/lib/ext.
